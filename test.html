oddOrEven

Should "odd" or "even" depending on the number
assert.equal(oddOrEven([0]), 'even');
assert.equal(oddOrEven([1]), 'odd')
assert.equal(oddOrEven([]), 'even')
Even tests
assert.equal(oddOrEven([0, 1, 5]), 'even')
assert.equal(oddOrEven([0, 1, 3]), 'even')
assert.equal(oddOrEven([1023, 1, 2]), 'even')
Negative Even tests
assert.equal(oddOrEven([0, -1, -5]), 'even')
assert.equal(oddOrEven([0, -1, -3]), 'even')
assert.equal(oddOrEven([-1023, 1, -2]), 'even')
Odd tests
assert.equal(oddOrEven([0, 1, 2]), 'odd')
assert.equal(oddOrEven([0, 1, 4]), 'odd')
assert.equal(oddOrEven([1023, 1, 3]), 'odd')
Negative Odd tests
assert.equal(oddOrEven([0, -1, 2]), 'odd')
assert.equal(oddOrEven([0, 1, -4]), 'odd')
assert.equal(oddOrEven([-1023, -1, 3]), 'odd')

vowels

Should return all valid vowels from a string

Should return an array of the vowels "e", "u", "i", "o", "o" from the string "The quick brown fox"
assert.deepEqual(vowels("The quick brown fox"), ["e", "u", "i", "o", "o"]);
Should return an array of the vowels "e", "o", "o" from the string "Hello World"
assert.deepEqual(vowels("Hello World"), ["e", "o", "o"]);

Should return all uppercase and lowercase vowels

Should return "A", "e", "E" from the string "cAse tEst"
assert.deepEqual(vowels("cAse tEst"), ["A", "e", "E"])
